# .JS
sort full content of the Async JavaScript as well as other core concept of the javascript via codes
## Installing

Using npm:

```bash
$ npm install axios
```

Using bower:

```bash
$ bower install axios
```

Using exress:

```bash
$ npm install express
```
Using ejs:

```bash
$ npm install ejs
```
Using UUID:

```bash
$ npm i uuid
```

Using yarn:

```bash
$ yarn add axios
```

Using jsDelivr CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
```

Using unpkg CDN:

```html
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
```
